# AgriAgents

A multi-agent AI system that helps farmers identify crop diseases, estimate affected area, predict disease spread risk, and generate actionable recommendations.

## Features

- Disease Detection
- Crop Identification
- Affected Area Estimation
- Weather-Aware Risk Prediction
- Context-Aware Recommendations
- PDF Reports
- Multi-Agent Architecture
- MCP Tools
